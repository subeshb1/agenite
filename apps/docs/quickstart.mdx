## Installation

First, install Agenite's core packages:

```bash
# Using npm
npm install @agenite/agent @agenite/tool

# Using yarn
yarn add @agenite/agent @agenite/tool

# Using pnpm
pnpm add @agenite/agent @agenite/tool
```

Then install your preferred LLM provider:

```bash
# For OpenAI
npm install @agenite/openai

# For Anthropic
npm install @agenite/anthropic

# For AWS Bedrock
npm install @agenite/bedrock

# For Ollama (local models)
npm install @agenite/ollama
```

## Basic example

Here's a simple example that creates an agent with a calculator tool:

```typescript
import { Agent } from '@agenite/agent';
import { Tool } from '@agenite/tool';
import { OllamaProvider } from '@agenite/ollama';

// Define calculator tool input type
interface CalculatorInput {
  operation: 'add' | 'subtract' | 'multiply' | 'divide';
  a: number;
  b: number;
}

// Create a calculator tool
const calculatorTool = new Tool<CalculatorInput>({
  name: 'calculator',
  description: 'Perform basic math operations',
  version: '1.0.0',
  inputSchema: {
    type: 'object',
    properties: {
      operation: { 
        type: 'string',
        enum: ['add', 'subtract', 'multiply', 'divide']
      },
      a: { type: 'number' },
      b: { type: 'number' },
    },
    required: ['operation', 'a', 'b'],
  },
  execute: async ({ input }) => {
    const { operation, a, b } = input;
    
    try {
      let result: number;
      switch (operation) {
        case 'add': result = a + b; break;
        case 'subtract': result = a - b; break;
        case 'multiply': result = a * b; break;
        case 'divide': 
          if (b === 0) throw new Error('Division by zero');
          result = a / b; 
          break;
      }
      
      return {
        isError: false,
        data: result.toString(),
      };
    } catch (error) {
      return {
        isError: true,
        data: error.message,
      };
    }
  },
});

// Create an agent
const agent = new Agent({
  name: 'math-buddy',
  provider: new OllamaProvider({ model: 'llama2' }),
  tools: [calculatorTool],
  instructions: 'You are a helpful math assistant.',
});

// Use the agent
const result = await agent.execute({
  messages: [
    { 
      role: 'user', 
      content: [
        { 
          type: 'text', 
          text: 'What is 1234 * 5678?' 
        }
      ] 
    }
  ],
});

console.log(result);
```

## Streaming responses

To get real-time responses from the agent:

```typescript
const iterator = agent.iterate({
  messages: [
    { 
      role: 'user', 
      content: [
        { 
          type: 'text', 
          text: 'Calculate 1234 * 5678 and explain the process' 
        }
      ] 
    }
  ],
  stream: true,
});

for await (const chunk of iterator) {
  switch (chunk.type) {
    case 'agenite.llm-call.streaming':
      // LLM is generating a response
      console.log('LLM:', chunk.content);
      break;
    case 'agenite.tool-call.params':
      // Agent is using a tool
      console.log('Tool:', chunk.toolUseBlocks);
      break;
    case 'agenite.tool-result':
      // Tool execution result
      console.log('Result:', chunk.result);
      break;
  }
}
```

## Adding state

Create a stateful agent that remembers previous calculations:

```typescript
interface CalculatorState {
  history: string[];
  total: number;
}

const agent = new Agent<CalculatorState>({
  name: 'stateful-calculator',
  provider: new OllamaProvider({ model: 'llama2' }),
  tools: [calculatorTool],
  initialState: {
    history: [],
    total: 0,
  },
  stateReducer: {
    history: (newValue, previousValue = []) => [...previousValue, newValue],
    total: (newValue, previousValue = 0) => previousValue + Number(newValue),
  },
  instructions: `You are a helpful math assistant that keeps track of calculations.
    Use the history and running total in your responses.`,
});
```
